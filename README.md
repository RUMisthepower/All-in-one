## 📦 О паке

Этот текстурпак - «сборная солянка» из лучших текстур 32×32, которыми я пользуюсь в своих мирах.  


### ✨ Особенности
- 🎨 Единый визуальный стиль при сохранении авторских особенностей
- 🧩 Модульность: можно использовать как полный пак, так и отдельные микро-паки

## 🤝 Как внести вклад

Хотите улучшить пак, добавить текстуры или исправить ошибку? 🎉

📖 Полная инструкция — в файле [**CONTRIBUTING.md**](./docs/CONTRIBUTING.md)

### Кратко:
1. Сделайте форк репозитория
2. Создайте ветку `feat/ваша-идея`
3. Внесите изменения и закоммитьте: `git commit -m "feat: добавил текстуры для Х"`
4. Отправьте ветку и создайте Pull Request

> ✨ Даже небольшие правки приветствуются!
